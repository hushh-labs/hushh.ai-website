# Apple Wallet Pass Images

This folder should contain the following images for your Hushh Social Card:

## Required Images

### Icon (Required)
- `icon.png` - 29×29 pixels
- `icon@2x.png` - 58×58 pixels  
- `icon@3x.png` - 87×87 pixels

### Logo (Required)
- `logo.png` - 160×50 pixels (max)
- `logo@2x.png` - 320×100 pixels (max)
- `logo@3x.png` - 480×150 pixels (max)

## Image Guidelines

1. **Format**: PNG with transparency
2. **Color**: Monochrome design (black/white/gray)
3. **Style**: Clean, minimal, Apple-like aesthetic
4. **Icon**: Should be the Hushh logo mark
5. **Logo**: Should be the Hushh wordmark

## Temporary Solution

For testing, you can create simple placeholder images:
- Use your existing Hushh logo from `public/Images/Logo/Logo.png`
- Resize to the required dimensions
- Ensure transparency is preserved

## Notes

- All images must be PNG format
- Retina images (@2x, @3x) are required for crisp display
- Images will be embedded in the .pkpass file
- Keep file sizes small for faster downloads
