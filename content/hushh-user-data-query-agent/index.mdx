## Architecture, Tips, and Story

- **Fan-Out Responses:** Besides rendering a conversational answer, the agent publishes the raw JSON onto an event bus so analytics, CRM, and marketing automation subscribe without extra API calls.
- **Latency Guardrails:** Requests exceeding 2 seconds trigger a lightweight fallback that returns previously cached context plus a notice about refresh status, keeping CS agents productive during spikes.
- **Prompt Libraries:** Team-specific prompt variants (support vs. marketing vs. growth) live in Supabase, letting each stakeholder fine-tune outputs without code changes.
